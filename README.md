# fresco
A proven and powerful methods to stabilize proteins.

**The main characteristics of FRESCO are (see figures below)**
- aim to obtain as many stabilizing mutations as possible
- proven to work on > 8 proteins
- high improvements in apparent melting temperature and much higher operational stability
- no signs of an activity-stability trade-off
- detailed 31 page manual written for those not experienced with computational work with the command line  
- experimentally, it requires screening one or a few 96-well plates
- it is a workflow requiring external software that need to be obtained via licenses. Specifically this version of fresco was tested with the latest versions of FoldX (2026), the latest version Yasara-Dynamics (25.12.2), and with an archive version of Rosetta (2015.25.20.57927) which is still available for download.

**Some hints**
- the manual is named protocolFRESCO<versionNumber>.pdf
- it is a good idea to test in advance if the targeted protein can be screened easily for thermostability. In our experience, for some 20% of the proteins standard high throughput assays like ThermoFluor fail.
